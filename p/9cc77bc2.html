树莓派入门01 | 安装系统

八月 18, 2021 391

拿到树莓派的第一步，装上不错的系统。

环境准备

  1. win10电脑

  2. 下载、安装好 SDCardFormatter、imager 软件

  3. micro-SD卡一张（32G 或 64G）

  4. 树莓派4B

关键步骤

  1. 格式化SD卡，用 SDCardFormatter 软件

  2. 烧录系统，用imager软件（树莓派官方）

  3. 选择好本地下载好的系统、SD卡的盘符，确认后点击 WRITE, 开始烧录进程

烧录树莓派系统

出现这一画面即为成功：

成功烧录系统

  1. 将SD卡插入树莓派，通电，开机

下载地址

  • SD官方组织：SD Memory Card Formatter | SD Association

  • imager 烧录软件：Raspberry Pi OS – Raspberry Pi

  • 系统镜像：Operating system images – Raspberry Pi
